**Тема:** Модульное тестирование и рефакторинг кода (TDD).

## Задание
1.  Провести рефакторинг кода РК1: разделить логику вычислений и вывод данных (print), чтобы функции возвращали значения.
2.  Написать модульные тесты (Unit Tests) для проверки основных функций задания (А1, А2, А3).
