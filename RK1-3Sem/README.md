# Рубежный контроль №1

## Вариант 29: Кафедра - Факультет

### Описание предметной области
- **Кафедра** (Department) - класс на стороне "много"
- **Факультет** (Faculty) - класс на стороне "один"

### Связи между классами
1. **Один-ко-многим**: Один факультет имеет несколько кафедр
2. **Многие-ко-многим**: Кафедра может принадлежать нескольким факультетам

### Выполненные задания

#### Задание А1
Список всех связанных кафедр и факультетов (один-ко-многим), отсортированный по факультетам.

#### Задание А2  
Список факультетов с средним бюджетом на кафедру, отсортированный по убыванию среднего бюджета.

#### Задание А3
Список всех кафедр, у которых в названии есть 'кафедра', и факультетов, с которыми они связаны.
